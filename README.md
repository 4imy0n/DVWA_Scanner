# 25년도 1학기 전공실무 프로젝트

## 프로젝트 소개
- DVWA를 타겟으로 한 취약점 스캐너 제작

## 프로젝트 기간
- 25.04.27 ~ 25.06.27

## 팀원 구성 및 역할
- 팀장 : 김우찬 - URL 크롤링, BruteForce, Command Injection, CSRF
- 팀원 : 이지민 - 디자인 설계, Stored XSS, Reflacted XSS, Dom XSS
- 팀원 : 우상범 - SQL Injection, File upload

## 개발환경
- VSCode
- Python

## 주요 기능
#### URL 크롤링
- DVWA URL 주소를 입력 시 크롤링을 통해 연결된 링크 모두 탐색

#### 취약점 자동 스캔
- 크롤링을 통해 얻은 URL로 총 9개의 취약점을 스캔
- 스캔 시 취약점이 존재한다면 취약점 발생 URL, 취약점 종류 확인 가능
- 옵션 기능을 통해 특정 URL 스캔, 전체 스캔, 특정 취약점 스캔, 스캔 결과 저장 가능

#### 취약점 자동 공격
- 취약점 스캔을 통해 얻은 취약 URL과 취약점 종류를 통해 해당 취약점 자동 공격 가능
